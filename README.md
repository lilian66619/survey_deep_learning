# 小组互评系统（GitHub版）

## 功能

- GitHub Pages 部署
- 在线问卷
- 自动提交到 GitHub Issues
- 教师后台
- 导出 CSV / Excel
- 手机扫码填写

---

# 第一步：创建 GitHub 仓库

例如：

group-survey

---

# 第二步：上传所有文件

包括：

- index.html
- style.css
- config.js
- app.js
- export.html
- export.js

---

# 第三步：创建 GitHub Token

打开：

https://github.com/settings/tokens

创建：

Classic Token

权限：

repo

---

# 第四步：修改 config.js

填写：

YOUR_GITHUB_USERNAME

YOUR_GITHUB_TOKEN

---

# 第五步：开启 GitHub Pages

Settings → Pages

Deploy from branch

main / root

---

# 学生填写网址

https://用户名.github.io/group-survey/

---

# 教师后台

https://用户名.github.io/group-survey/export.html

---

# 数据后台

GitHub 仓库 → Issues
